webhooks plugin for `Tutor <https://docs.tutor.edly.io>`__
##########################################################

webhooks plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/aulasneo/tutor-contrib-webhooks.git

Usage
*****

.. code-block:: bash

    tutor plugins enable webhooks
